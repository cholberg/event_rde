## Installation

Clone the repository:

```
git clone https://github.com/cholberg/event_rde
```

Install:

```
cd event_rde
pip install .
```

Make sure to have jax installed. For CPU:

```
pip install "jax[cpu]"
```

For NVIDIA GPU:

```
pip install -U "jax[cuda12_pip]" -f https://storage.googleapis.com/jax-releases/jax_cuda_releases.html
```
